## How to start
Installing requirements:

    pip install -r requirements.txt


Starting BlockChain:

    chmod +x Blockchain.py
    python Blockchain.py


Starting Miner:
    chmod +x miner.py  
    python miner.py
    

## Only works with Python 2.6 >

I recommend creating an virtual environment with anaconda
